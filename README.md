# Portfolio - Apimef

This repository contains the **Portfolio Website** for **Apimef**, showcasing my family's honey business and my web development skills.  
The focus of this project was to create a professional, responsive, and visually appealing portfolio website.

## Project Overview
- Showcase of Apimef honey products  
- Sections included: Home, Gallery, Shop, Login, Contact, About / Why this project  
- Links to GitHub, Holberton, LinkedIn  
- Responsive design for desktop and mobile devices  

## Tech / Stack
- HTML5 / CSS3  
- JavaScript  
- Node.js / Express  
- MySQL / MongoDB (backend for shop functionality)  
- Docker (for deployment)  
- Git for version control  

## What I Learned
- Full-stack web development  
- Responsive design and UX best practices  
- Connecting frontend to backend with Node.js and databases  
- Deploying projects using Docker  
- Showcasing a professional project for portfolio purposes  

## Usage
Clone the repository and follow the instructions to run the website locally using Node.js and Docker, or open the HTML files for static viewing.
